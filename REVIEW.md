# 검토 메모

## 디자인 검토

- 실제 교회 홈페이지에서 많이 쓰이는 구조인 상단 메뉴, 사진형 첫 화면 안내, 예배 시간, 교회 소개, 사역, 설교, 오시는 길 순서로 구성했습니다.
- 과한 최신 유행 효과 대신 밝은 종이색, 청록, 남색, 골드 계열을 사용해 차분하고 익숙한 분위기로 정리했습니다.
- 카드 모서리와 그림자는 과하지 않게 적용했습니다.

## 모바일 검토

- 920px 이하에서 모바일 메뉴로 전환됩니다.
- 640px 이하에서 안내 박스, 예배 카드, 사역 카드가 1열로 바뀝니다.
- 모바일에서는 히어로 사진 초점, 제목 크기, 버튼 배치를 따로 조정했습니다.
- 긴 한국어 문장이 좁은 폭에서 넘치지 않도록 줄바꿈을 보강했습니다.

## 접근성 검토

- `lang="ko"`, 본문 바로가기 링크, 명확한 섹션 구조를 넣었습니다.
- 모바일 메뉴 버튼에 `aria-expanded`, `aria-controls`, `aria-label`을 적용했습니다.
- 로고 SVG에는 제목과 설명을 포함했습니다.

## 라이선스 검토

- 외부 이미지, 외부 폰트, CDN, 아이콘 라이브러리를 사용하지 않았습니다.
- 로고와 약도는 직접 작성한 SVG입니다.
- 첫 화면 교회 건물 사진은 외부 사이트에서 가져온 사진이 아니라 이 프로젝트 용도로 새로 생성한 이미지입니다.
- 실제 운영 전에 교회명, 주소, 전화번호, 이메일만 실제 정보로 교체하면 됩니다.
