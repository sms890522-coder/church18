# 은혜샘교회 홈페이지

정적 HTML/CSS/JavaScript로 만든 교회 홈페이지입니다.

## 파일

- `index.html`: 페이지 구조와 콘텐츠
- `styles.css`: 반응형 디자인
- `script.js`: 모바일 메뉴 동작
- `assets/logo.svg`: 자체 제작 SVG 로고
- `assets/hero-church.png`: 생성형 이미지로 만든 교회 건물 히어로 배경

## 라이선스 메모

외부 이미지, 외부 폰트, 아이콘 라이브러리, CDN을 사용하지 않았습니다. 로고와 약도는 프로젝트 안에서 직접 작성한 SVG이고, 첫 화면 교회 건물 배경은 이 프로젝트 용도로 새로 생성한 이미지입니다.
